# Conversational AI

Chatbots, voice assistants, and multi-turn dialogue systems built on **Amazon Bedrock** and **Amazon Lex**, with persistent memory and real-time transcription.

## Applications

### [Meeting Context Navigator](./meeting-context-navigator/)
Index meeting transcripts and query them conversationally. Retrieve decisions, action items, and key discussions in seconds — no more scrubbing through hour-long recordings.

**Key AWS services:** Bedrock (Claude), Transcribe, S3, DynamoDB, OpenSearch Serverless

**Use cases:** Post-meeting follow-up, async team collaboration, executive briefing prep, compliance audit trails
